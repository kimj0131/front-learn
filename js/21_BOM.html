# Browser Object Model

  • 웹 브라우저를 구조화해놓은 객제
  • window 객체를 통해 웹 브라우저를 직접 제어할 수 있다
  • 자바스크립트에서 window는 생략이 가능하다

# BOM의 객제들

  • window : 웹 브라우저 최상위 객체
  • window.document : 현재 웹 페이지의 HTML코드 구조 객체(DOM)
  • window.screen : 모니터 등 기기 화면에 대한 정보
  • window.history : 웹 브라우저의 페이지 이동 내역
  • window.location : 현재 웹 브라우저가 위치한 경로에 대한 정보
  • window.navigator : 웹 브라우저에 대한 버전 정보
