# Web Storage API

  • 웹 브라우저에 데이터를 보관할 수 있는 공간
  • 데이터를 key-value 형식으로 저장할 수 있다
  • 로컬 스토리지는 웹 브라우저를 종료하더라도 데이터가 유지된다
  • 세션 스토리지는 웹 브라우저를 종료하면 데이터가 세션과 함께 만료된다
    (세션 : 서버와 연결이 유지되고 있는 상태, ex: 은행 사이트 로그인)

# 1 to 50 만들기

  • 게임이 끝날 때 전적을 로컬스토리지에 기록하기
  • 로컬스토리지에 저장되어있는 랭킹을 순서대로 출력 (방식은 마음대로)
