# 링크를 통해 다른 페이지로 이동하기

1. a 태그를 이용하기
2. 자바스크립트 이용하기

# a(anchor) tag

- a 태그는 전체 글의 일부에 링크를 걸 수 있는 태그이다
- 자신의 영역을 생성하는 태그가 아니다.
- href 속성에 가고싶은 곳의 주소(url)을 설정해 놓는다
- 절대 경로와 상대 경로를 모두 사용할 수 있다

네이버로 가기
네이버로 가기(http를 뺌)
네이버로 가기(http://를 뺌)
Google로 가기

# URL 경로의 종류

- naver.com : 상대경로(현재 위치)
- ./naver.com : 위와 같은 것
- /naver.com : 도메인부터 시작되는 상대경로
- //naver.com : 프로토콜부터 시작되는 상대경로
- https://naver.com : URL의 절대 경로

- 요소
