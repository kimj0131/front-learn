# 웹 페이지에 표 출력하기

- <table> : 표를 의미하는 태그
- <tr> : table row, 표의 한 행을 의미하는태그
- <th> : table header, 한 행에 제목 칸을 추가하는 태그
- <td> : table data, 한 행에 일반 칸을 추가하는 태그

- colspan attr : 해당 칸이 몇 열을 차지하는지 설정하는 속성
- rowspan attr : 해당 칸이 몇 행을 차지하는지 설정하는 속성
