# Position property

  • 포지션 속성을 설정한 요소는 포지션이 설정되지 않은 요소들을 무시하면서 자유롭게 요소를 배치할 수 있다
  • static(default) - 포지션이 설정되지 않은 상태
  • absolute - 절대적으로 위치를 설정
  • relative - 상대적으로 위치를 설정
  • fixed - 화면에 고정할 위치를 설정
  • sticky - 스크롤이 변하면서 고정될 위치를 설정
  • 포지션이 겹치는 요소가 있다면 z-index 속성으로 어떤 요소가 위에 있을지 설정할 수 있다
  • 기본 z-index는 0이고 기본 요소보다 밑에 깔리도록 설정하고 싶다면 z-index를 음수로 설정할 수도 있다

# position: absolute

  • 부모 요소 내부에서 절대적인 위치를 설정
  • 만약 부모 요소에 포지션이 설정되어 있지 않다면 더 위의 조상으로 거슬러 올라간다
  • 포지션을 설장하고 나면 top, left, bottom, right를 통해 위치를 지정할 수 있다
